Для генерации базы данных используются утилиты
 - из cygwin
 - m4 (препроцессор)
 - psql (интерпретатор PG SQL)

1.0 Скачать и установить cygwin
    - 32-битные бинарники вполне работают на 64-битной Windows

1.1 в переменную окружения PATH добавить путь к каталогу cygwin\bin

2.0 в переменную окружения PATH добавить путь к каталогу PostgreSQL\<номер версии>\bin
    - нужно что бы psql.exe мог запускаться без явного указания пути

3.0 скачать и скопироать в cygwin\bin
     m4.exe
     regex2.dll
     
     либо из internet, либо из каталога distr

     можно скопировать в любой другой каталог, который перечислен в PATH
     
4.0 Привязать расширение .sh к bash.exe из каталога cygwin\bin

Теперь можно запускать из командной строки (cmd.exe) init.sh для генерации базы данных
