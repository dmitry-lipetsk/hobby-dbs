Windows

1. По умолчанию PG ставится под акаунтом "Network Services"

   Путь к каталогу с базами данных должен быть доступен на чтение/запись

2. После установки, в настройках сервиса можно указать другой аккаунт

   Проверить, что новому аккаунту доступен каталог с базами данных на чтение/запись

3. ПРОБЛЕМЫ С ОТОБРАЖЕНИЕМ ОШИБОК В pgAdmin/консоли (ни не выводятся)

   В каталоге с базами данных найти и открыть файл postgresql.conf

   Указать lc_messages = 'UTF8'
    - по умолчанию там что-то типа lc_messages = 'Russian_Russia.1251'
